在微软拼音输入法中直接输入Unicode码

插入基于拉丁语的 ASCII 或 Unicode 符号或字符 - Microsoft 支持

首先键入十六进制数字（将其插入文档中），然后键入Alt+X将其更改为匹配的Unicode字符。

方法：先输入“VUC”（不区分大小写），然后用十六进制输入Unicode编码。

示例：

  • “🐎”（马的Emoji表情符号），Unicode编码U+1F40E，在微软拼音输入法中输入“VUC1F40E”，得到“🐎”字符。
  • “𪠽”（噹的类推简化字），Unicode编码U+2A83D，在微软拼音输入法中输入“VUC2A83D”，得到“𪠽”字符。

简析Windows注册表

五大主键的作用

每个注册表都有上图所示的五个主键，以HKEY开头，这五个主键的含义分别为：

  • HKEY_CLASSES_ROOT： 包含文件扩展名和文件类型，还包括我的电脑、回收站、控制面板等的类标识
  • HKEY_CURRENT_USER：保存有当前登录的用户的配置信息
  • HKEY_LOCAL_MACHINE：包含本机上所有软硬件的信息
  • HKEY_USERS：保存着所有本机上用户的信息。包括通用设置(如应用程序事件)以及特定用户的设置(如桌面)
  • HKEY_CURRENT_CONFIG：包含所有连结到此机上的硬件的配置数据,如打印机和显示器的配置数据

UUID 和 GUID

  • UUID 是一种通用唯一识别码，重复的概率接近零的通用协议。GUID是MS公司对UUID 协议的实现
  • windows系统中多个地方都使用了GUID
    • 通用唯一识别码 - 维基百科，自由的百科全书
    • Windows 此电脑-＞设备与驱动器-＞图标管理_设备和驱动器中怎么添加图标呢-CSDN博客
    • 关于 设备安装类GUID、设备接口类GUID_classguid guid-CSDN博客
    • 在windows操作系统环境下，设备通常被当作特殊文件处理。要打开设备，就要知道该设备的路径，要找到设备的路径，要使用GUID来查找。
